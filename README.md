📋 Job Application Tracker

A simple and efficient Job Application Tracker built to help you organize, monitor, and manage your job search in one place. Stop losing track of applications in spreadsheets and keep everything structured, searchable, and easy to update.

✨ Features
🧭 Track job applications in one dashboard
🏢 Store company names, job titles, and application links
📊 Monitor application status (Applied, Interviewing, Offer, Rejected, etc.)
📝 Add notes for interviews, follow-ups, and reminders
🔎 Easily search and filter applications
📅 Keep your job search organized and up to date
🚀 Why I Built This

Job hunting can quickly become overwhelming—especially when applying to multiple roles across different platforms. This project was built to:

Eliminate messy spreadsheets
Provide a clear overview of all applications
Make it easier to track progress and follow-ups
Help stay consistent and organized during the job search

⚙️ Installation & Setup
# Clone the repository
git clone https://github.com/EllisHeidi/Job-Application-Tracker.git

# Navigate into the project folder
cd Job-Application-Tracker

# Install dependencies
npm install

# Start the development server
npm start
📌 Usage
Add a new job application with company and role details
Update the status as you progress through the hiring process
Add notes for interviews, emails, or follow-ups
Filter or search to quickly find applications
